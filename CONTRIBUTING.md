# Contributing to JigglyMouser

Thank you for considering contributing!

## How to Contribute

1. Fork the repo
2. Create a feature branch
3. Make your changes
4. Test that it compiles with `build.bat` or CMake
5. Submit a pull request

## Building

See the README for build instructions.

## Reporting Bugs

Open an issue and include:
- Your operating system
- Steps to reproduce
- Expected vs actual behavior

## Suggestions

Feature requests are welcome! Open an issue to discuss.
> Make sure to tag your requests.
